\emph{Howto}:

Um das Zertifikat zu ändern, wird der Untermenüpunkt "Zertifikat ändern" des
Menüs "Einstellungen" gewählt. Wird versucht, sich auf einem https-Server
einzuloggen, ohne das dieser Server dem System bekannt ist, erscheint ein
Dialog, der ebenfalls ermöglicht, dass eine neue Zertifikatdatei gesetzt
wird.

Es öffnet sich ein Dateiauswahlfenster in dem die entsprechende Datei
ausgewählt werden kann.
