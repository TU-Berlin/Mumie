\emph{Howto}:

Um das * zu testen, ist folgendes zyklisch zu tun:
\begin{enumerate}
\item Den Graph speichern.
\item Prüfen, dass das * nicht angezeigt wird.
\item Eine Veränderung durchführen.
\item Prüfen, dass das * angezeigt wird.
\end{enumerate}


Zu Testen ist:
\begin{itemize}
\item Einfügen von Komponenten
\item Verschieben von Komponenten 
\begin{itemize}
  \item mit Maus und
  \item mit Tastatur in alle vier Richtungen
\end{itemize}
\item Einfügen von Subkomponenten
\item Verbinden von Komponenten
\begin{itemize}
  \item automatisch
  \begin{itemize}
    \item mit der entsprechenden Taste
    \item mit dem Button in der Toolbar
    \item mit dem Kontextmenu
  \end{itemize}
  \item von Hand
\end{itemize}
\item Ändern der Richtung einer Kante
\begin{itemize}
  \item mit der entsprechenden Taste
  \item Mit dem Button aus der Toolbar
  \item mit dem Kontextmenü
\end{itemize}
\item Verändern von Kanten
\begin{itemize}
  \item Verändern von Kantenteilstücken
  \item Verändern von ganzen Kanten
\end{itemize}
\item Löschen von Komponenten
\begin{itemize}
  \item mit der entsprechenden Taste
  \item Mit dem Button aus der Toolbar
  \item mit dem Kontextmenü
\end{itemize}
\item Erstellen des Roten Faden 
\begin{itemize}
  \item Ziehen einer roten Kante mit dem Kontextmenü
  \item Ändern der Farbe einer Kante
  \begin{itemize}
    \item mit den entsprechenden Tasten
    \item Mit den Buttons aus der Toolbar
    \item mit dem Kontextmenü
  \end{itemize}
  \item Erzeugen eines generischen Roten Faden
\end{itemize}
\item Zuweisen eines Dokumentes
\item Zuweisen einer Summary
\item Setzen oder löschen von Punkten
\item Setzen eines Labels
\item Ändern der Metainformationen
\begin{itemize}
  \item der Name
  \item die Beschreibung
  \item der Status
  \item die Lehrveranstaltung
  \item die Kategorie
  \item der Bearbeitungszeitraum
\end{itemize}
\end{itemize}
