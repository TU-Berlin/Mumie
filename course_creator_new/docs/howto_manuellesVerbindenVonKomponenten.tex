\emph{Howto}:

Es wird eine Graphkomponente mit der rechten Mausetaste
angeklickt und im erscheinenden Kontextmenü der Unterpunkt ''Schwarze Kante
ziehen'' gewählt. Dann werden mit der linken Maustaste beliebige Eckpunkte der
Kante festgelegt. Das jeweils zuletzt erzeugte Kantenstück kann mit der rechten
Maustaste wieder gelöscht werden. Ganz gelöscht wird die Kante, wenn wieder auf
die Startkomponente geklickt wird. Beendet wird die Kante durch Klicken auf
eine zweite Graphkomponente.
