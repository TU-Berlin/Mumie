\emph{Test}:
\begin{itemize}
\item Das Dialogfenster öffnet sich, wenn der Untermenüpunkt "Label setzen"
  aus dem Kontextmenü gewählt wird.
\item Beim Öffnen des Dialogfensters steht dort das zuletzt angegebene Label.
\item beim Beenden des Dialogs mit "Cancel" wird der angegebene String nicht
  übernommen.
\item beim Beenden des Dialogs mit "Ok" wird der angegebene String übernommen.
\item Es ist nicht möglich, das Label leer zu lassen.
\item Das Defaultlabel ist "- no label -" (ggf prüfen durch Speichern).
\item Nur Dokumentkomponenten können Label zugewiesen werden. Branchkomponenten
  und Kanten haben keine.
\end{itemize}
