\emph{Howto}:

Zwei Komponenten werden wie folgt vertauscht:
\begin{itemize}
\item Mit der Tastatur vertauscht werden können zwei ausgewählte Knoten wenn
  die Taste, die dafür in der Tastenkonfiguration eingetragen ist, gedrückt
  wird.
\item Zwei ausgewählte Komponenten werden mit dem Vertauschen-Button
  \image{howtopics/tool_change.png} verbunden.
\item Sind zwei Komponenten ausgewählt, so erscheint im Kontextmenü (rechte
Maustaste) der Punkt ''Vertauschen''.
\end{itemize}
