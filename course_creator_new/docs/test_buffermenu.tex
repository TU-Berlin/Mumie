\emph{Test}:
\begin{itemize}
\item Die Buffereinträge haben die Form "Name des Graphen" - "Dateiname"
  gegebenenfalls gefolgt von einem *.
\item Ist ein Graph noch nicht gespeichert, so erscheint statt des Dateinamens
  die Auskunft "kein Dateiname".
\item Der Name wird korrekt angezeigt und wird geändert, wenn er im Eingabefeld
  des Metainformationsfensters geändert wird.
\item Die Einträge habe die Reihenfolge in der sie zuletzt angezeigt wurden.
\item Wird mit Hilfe des Buffermenüs ein anderer Graph angezeigt, so ändert
  sich der angezeigte Graph entsprechend, aber es werden auch die
  entsprechenden Metainformationen im Metainformationsfenster angezeigt.
\end{itemize}
