Der Benutzer kann entscheiden, ob er
\begin{itemize}
\item (linker Button) die Datei öffnen möchte und deren Dateiname löscht. Der Versuch, die
  Datei zu speichern, führt dann automatisch zu einem "Speichern unter".
\item (mittlerer Button) die Datei öffnen möchte und den bereits geöffneten Graphen löscht (alle
  Änderungen gehen verloren) oder
\item (rechter Button) das Laden abbricht.
\end{itemize}
