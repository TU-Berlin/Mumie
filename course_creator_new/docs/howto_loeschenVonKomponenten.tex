\emph{Howto}:

Das Löschen geschieht auf unterschiedliche Art:

\begin{itemize}
\item Durch Auswählen des Punktes "Löschen" im Kontextmenü der markierten Komponente[n].
\item Durch Anklicken des Löschenbuttons
  \image{howtopics/tool_delete_komp.png} in der Toolbar.
\item Durch Drücken der Taste, die dafür in der Tastenkonfiguration eingetragen
  ist.
\end{itemize}
