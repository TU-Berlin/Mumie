\emph{Howto}:

Um dIE Summary zu ändern muss das Elementzuweisungsfenster geöffnet
werden.Siehe \href{\#elementzuweisungsfenster}{Elementzuweisungsfenster}. Im
Elementzuweisungsfenster wird dann eine Summary aus dem Baum ausgewählt und
der "Set"-Button gedrückt. Der Dateiname der Summary wird angezeigt. Ist dieser
mit Pfad zu lang für das Feld, so wird der Pfad abgekürzt. Der vollständige
Pfad erscheint als Tooltip.
