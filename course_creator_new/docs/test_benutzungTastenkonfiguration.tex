\emph{Test}:
\begin{itemize}
\item Auf die eingetragenen Zeichen reagiert das Programm in der gewünschten
  Weise. (eine Taste muss nur für eine Funktion getestet werden, insgesamt
  sollen aber alle Funktionen wenigstens einmal getestet werden.)
\item Zeichen dürfen sich nicht doppeln. Gegebenenfalls erscheint eine
  Mitteilung und man kann auswählen, ob man dieses Tastaturkürzel doch benutzen
  möchte und das andere löscht, oder ein neues Zeichen eingeben möchte.
\item Die Modifier Shift, Ctrl und Alt sind möglich und werden in das
  Eingabefenster geschrieben.
\item Der Defaultbutton setzt nach einer "wollen Sie wirklich?"-Abfrage die Eingaben zurück auf die Defaultwerte.
\item Die Eingaben sind sofort verwendbar.
\item Ist eine Taste nicht möglich, erscheint eine Fehlermeldung.
\end{itemize}
