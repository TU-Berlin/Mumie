\emph{Test}:
\begin{itemize}
\item Das Ändern der Debugeinstellungen im Debugfenster wird in die
  Konfigurationsdatei übernommen.
\item Die "Häkchen" im Debugfenster sind so gesetzt, wie sie in der
  Konfigurationsdatei stehen.
\item Nur wenn eine Variable "true" ist und die "DEBUG"-Variable ebenfalls
  "true" ist, werden Debugausgaben getätigt.
\item Ist die Exception-Variable "true", so werden auftretende Exceptions als
  Dialog ausgegeben (ansonsten gar nicht).
\end{itemize}
