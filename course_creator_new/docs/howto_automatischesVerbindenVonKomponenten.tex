\emph{Howto}:

Um die zwei markierten Knoten zu verbinden gibt es folgende Möglichkeiten:
\begin{itemize}
\item Mit der Tastatur verbunden werden können zwei ausgewählte Knoten, wenn die
  Taste, die in der Tastenkonfiguration dafür eingetragen ist, gedrückt wird.
\item Zwei ausgewählte Komponenten werden mit dem Verbinden-Button
  \image{howtopics/tool_edge_connect.png} verbunden.
\item Sind zwei Komponenten ausgewählt, so erscheint im Kontextmenü (rechte
  Maustaste) der Punkt ''Verbinden''.
\end{itemize}
