\emph{Howto}:

Ein neuer Graph wird erstellt, indem die Taste, die dafür in der
Konfigurationsdatei eingetragen ist, gedrückt wird, oder der Untermenüpunkt
"Neu" im Menüpunkt "Datei" gewählt wird. Ebenso kann der
\image{howtopics/tool_only_new.png} Button aus der toolbar gewählt werden.

zur Vereinfachung wurde eine Abfragefolge angelegt, so dass das erzeugen eines
neuen Kurses übersichtlicher wird. In der folgenden Graphik ist bereits ein in
zukunft zu implementierender Graphtyp angezeigt, dieser existiert aber noch
nicht.
