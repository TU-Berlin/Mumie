\emph{Howto}:

Um eine rote Kante zu generieren, werden die schwarzen Kanten
markiert und dann
\begin{itemize}
\item im Kontextmenü dieser Kanten der Punkt "erzeuge Roten Faden" gewählt
\item der Button \image{howtopics/tool_edge_red.png} aus der Toolbar
  angeklickt
\item die Taste, die in der Tastenkonfiguration dafür eingetragen ist, drückt.
\end{itemize}
wird.
